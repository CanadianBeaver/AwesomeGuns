Extensions and compatible mods

Awesome Guns modification is a FULL modification. It works fine on Windows and Linux without UFOextender and XcomUtil. Also, Awesome Guns modification works without problems on the OpenXcom 1.0 (milestones) and on each version of the OpenXcom nightly. Unfortunately, there is a difference between OpenXcom 1.0 and OpenXcom nightly. This difference can be resolved by extensions which have been developed for specific version of OpenXcom. Also, there are some modifications for OpenXcom which are compatible with Awesome Guns modification.

This page describes about extensions and compatible mods for the Awesome Guns modification.

Geoscape cities and countries extension

Many cities have been added to all regions of the globe. This will make the game more interactive, especially during alien terror missions.

Also, the borders between countries and regions have been corrected. For example: Kaliningrad and Vladivostok are related to Russia, Toronto is related to Canada, Honolulu is related to the USA.

This extension can be downloaded from web page.

Compatible mods

Awesome Guns modification is a FULL modification. It works fine on Windows and Linux without UFOextender and XcomUtil. There are a few mods for improving alien behaviour and they work fine with my creation. I recommend to use these mods:

  • Lukes Expanded Terror Mission
  • Solar's New UFOs (be careful, only version 1.1 is for stable game's release)
