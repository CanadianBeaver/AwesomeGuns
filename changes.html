Additional changes

Awesome Guns modification implements a lot of minor changes. These changes makes the OpenXcom game more interesting and more exciting. Some of changes improved the game and corrected unimportant mistakes, some of changes have been implemented just for fun. The following are the most interesting additional innovations.

Geoscape cities

Many cities have been added to the globe in all regions. This will make the game more interesting.

Have you ever played DOOM II?

It is funny, is not it?

Говорим по-русски

Модификация УдивительныеПушки понимай и говори на твой язык скифов. Игра полностью переведена, а некоторые тексты адаптированы с учётом специфики русского языка.

Battlescape control panel

Just a small change for a better view during combats. Buttons on the Control panel have been colored.

Alien reproduction

Access to the alien reproduction is opened. The research and ufopaedia side were existed in the OpenXcom game.

Aircraft ramp

The walk-in time on ramp of aircrafts has been reduced to prevent jumping off from the ramp.
