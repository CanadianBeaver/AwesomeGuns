Additional changes

Awesome Guns modification implements a lot of minor changes. These changes makes the OpenXcom game more interesting and more exciting. Some of changes improved the game and corrected unimportant mistakes, some of changes have been implemented just for fun.

Geoscape cities

Many cities have been added to the globe in all regions. This will make the game more interesting.
